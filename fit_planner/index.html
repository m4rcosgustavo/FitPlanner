SOBRE O PROJETO

O FitPlanner é um projeto acadêmico feito para praticar desenvolvimento web, criando uma solução simples para organizar treinos de academia.

Aqui você consegue registrar exercícios, séries, repetições e cargas e ter uma visão melhor da sua evolução.

Contexto Acadêmico

Projeto desenvolvido para aprender na prática como funciona um app web com HTML, CSS e JavaScript.

Tecnologias Utilizadas

  • HTML
  • CSS
  • JavaScript
