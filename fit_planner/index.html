SOBRE O PROJETO

O FitPlanner é um aplicativo web desenvolvido como projeto acadêmico, com o objetivo de aplicar conceitos iniciais de programação web na construção de uma solução funcional voltada ao planejamento de treinos de academia.

A aplicação permite que o usuário organize exercícios, séries, repetições e cargas, além de acompanhar sua evolução de forma simples e acessível.

Contexto Acadêmico

Projeto desenvolvido para fins de prática e consolidação de conceitos básicos de desenvolvimento web.

  • CSS
  • JavaScript
